Tipos de selectores que entran en examen

  1. Clase (class=””)

    2. El selector "clase" selecciona elementos con un atributo de clase específico. Para seleccionar elementos con una clase específica, se escribe un carácter de punto (.) seguido del nombre de la clase. También puede especificar que solo los elementos HTML específicos se vean afectados por una clase. Para hacer esto, se comienza con el nombre del elemento,y luego se escribe el carácter de punto (.), seguido del nombre de la clase

    Ej.: El otro día vi un partido de fútbol del Barcelona.

  2. Identificador (id=””)

    3. El id atributo especifica una identificación única para un elemento HTML (el valor debe ser único dentro del documento HTML). CSS y JavaScript pueden usar el valor de identificación para realizar ciertas tareas para el elemento con el valor de identificación específico. En CSS, para seleccionar un elemento con una identificación específica, se debe escribir un carácter hash (#), seguido de la identificación del elemento:

  3. Todos los elementos (*)

    4. El selector universal funciona como un carácter comodín, seleccionando todos los elementos en una página. Cada página HTML se basa en contenido colocado dentro de etiquetas HTML. Cada conjunto de etiquetas representa un elemento en la página.
